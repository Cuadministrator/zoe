页面模板 方便快速构建页面


## API

属性 | 说明 | 类型 | 默认值
----|-----|------|------
| showErrorPage   | 是否展示错误页面 | boolean | false |
| errorConfig   | ResultPage 组件配置 | object | 无 |
| style   | 展示的最小高度 | object（样式） | 无 |
| header   | 头部元素 | React.Element（Header组件） | 无 |
| footer   | 底部元素 | Footer组件 | 无 |
| children   | 子元素 | React.Element | 无 |
